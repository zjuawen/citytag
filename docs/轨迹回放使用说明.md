# 高德地图轨迹回放工具使用说明

## 📋 功能说明

这个工具可以将 `history_tracks.json` 文件中的轨迹点在高德地图上按时间顺序回放显示。

## 🚀 快速开始

### 1. 运行程序

```bash
python3 replay_tracks_gaode.py
```

或者指定JSON文件路径：

```bash
python3 replay_tracks_gaode.py history_tracks.json
```

### 2. 获取高德地图API Key

**重要：** 程序需要高德地图API Key才能正常显示地图。

#### 申请步骤：

1. 访问高德开放平台：https://console.amap.com/dev/key/app

2. 注册/登录账号

3. 创建应用：
   - 点击"创建新应用"
   - 填写应用名称（如：轨迹回放工具）
   - 选择应用类型：Web端（JS API）

4. 添加Key：
   - 在应用下点击"添加"
   - Key名称：轨迹回放
   - 服务平台：Web端（JS API）
   - 白名单：可以留空（开发测试）或添加域名限制

5. 复制生成的Key

### 3. 配置API Key

打开生成的 `track_replay_gaode.html` 文件，找到这一行：

```html
<script src="https://webapi.amap.com/maps?v=2.0&key=YOUR_API_KEY&callback=initMap"></script>
```

将 `YOUR_API_KEY` 替换为你申请到的API Key。

### 4. 打开HTML文件

直接在浏览器中打开 `track_replay_gaode.html` 文件即可。

## 🎮 功能说明

### 控制按钮

- **▶ 开始回放**：开始按时间顺序播放轨迹点
- **⏸ 暂停**：暂停当前回放
- **⏹ 停止**：停止回放
- **↺ 重置**：重置到开始位置

### 回放速度

可以选择不同的回放速度：
- 0.5x：慢速回放
- 1x：正常速度（默认）
- 2x：2倍速
- 5x：5倍速
- 10x：10倍速

### 信息面板

右侧信息面板显示：
- **序号**：当前轨迹点序号 / 总点数
- **时间**：轨迹点的时间戳
- **纬度/经度**：当前位置坐标
- **精度**：GPS精度（米）
- **电量**：设备电量百分比
- **进度条**：回放进度

## 📊 数据格式要求

JSON文件应包含以下结构：

```json
{
  "device_info": {
    "sn": "设备序列号",
    "name": "设备名称"
  },
  "query_info": {
    ...
  },
  "track_points": [
    {
      "latitude": 26.09117,
      "longitude": 119.25496,
      "timestamp": "2026-02-24 18:24:21",
      "accuracy": "113",
      "batteryLevel": 100
    }
  ]
}
```

## ⚠️ 注意事项

1. **API Key限制**：
   - 免费版API Key有调用次数限制
   - 个人开发者每天有免费额度
   - 超出限制后地图可能无法正常显示

2. **浏览器兼容性**：
   - 推荐使用 Chrome、Firefox、Edge 等现代浏览器
   - 需要支持JavaScript

3. **数据量**：
   - 轨迹点数量过多可能影响性能
   - 建议单次回放不超过1000个点

4. **网络要求**：
   - 需要联网加载高德地图API
   - 首次加载可能需要一些时间

## 🔧 故障排除

### 问题1：地图不显示

**可能原因：**
- API Key未配置或配置错误
- API Key已过期或被禁用
- 网络连接问题

**解决方法：**
1. 检查API Key是否正确配置
2. 检查浏览器控制台是否有错误信息
3. 确认网络连接正常

### 问题2：轨迹点不显示

**可能原因：**
- JSON文件格式不正确
- 轨迹点数据为空
- 坐标数据格式错误

**解决方法：**
1. 检查JSON文件格式是否正确
2. 确认 `track_points` 数组不为空
3. 确认 `latitude` 和 `longitude` 字段存在且为数字

### 问题3：回放不流畅

**可能原因：**
- 轨迹点数量过多
- 浏览器性能问题
- 回放速度过快

**解决方法：**
1. 降低回放速度
2. 关闭其他占用资源的标签页
3. 使用性能更好的浏览器

## 📝 技术说明

- **地图API**：高德地图 JavaScript API v2.0
- **数据格式**：JSON
- **回放方式**：按时间顺序逐点显示
- **轨迹线**：自动绘制完整轨迹路径
