# CityTag 轨迹查询工具 - 快速参考

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install pycryptodome requests
```

### 2. 快捷查询（推荐）
```bash
python3.11 query_tracks.py quick
```
或
```bash
python3.11 query_tracks.py
# 然后输入 y
```

**功能：** 自动查询最近3天数据，保存到 `history_tracks_3days.json`

## 🔑 Token 管理

**自动保存：** 登录后 token 保存在 `.citytag_token.json`

**清除 Token：**
- 删除 `.citytag_token.json` 文件
- 或在交互式查询结束时选择清除

## ❓ 常见问题

**Q: Token 过期怎么办？**  
A: 删除 `.citytag_token.json`，重新运行脚本登录

**Q: 没有找到轨迹点？**  
A: 尝试更大的时间范围，检查设备序列号

**Q: 数据重复？**  
A: 使用合并功能会自动去重

**Q: 查询速度慢？**  
A: 使用快捷查询模式，或减小时间范围
