# 获取历史轨迹点说明

## 概述

要获取设备的历史多个轨迹点，需要提供设备序列号（sn）和时间范围（startTime 和 endTime）。

## 请求参数

### 必需参数

- `token`: 用户 token（用于加密）
- `uid`: 用户 ID
- `sn`: 设备序列号

### 时间范围参数（获取历史轨迹点时必需）

- `start_time`: 开始时间戳（**毫秒**）
- `end_time`: 结束时间戳（**毫秒**）

## 请求示例

### Python 代码示例

```python
from generate_request_body import call_and_decrypt_api, get_token_and_uid
from datetime import datetime, timedelta

# 1. 获取 token
token, uid = get_token_and_uid()

# 2. 设置时间范围（例如：最近24小时）
end_time = int(datetime.now().timestamp() * 1000)  # 当前时间（毫秒）
start_time = end_time - 24 * 60 * 60 * 1000  # 24小时前（毫秒）

# 3. 发送请求获取历史轨迹点
result = call_and_decrypt_api(
    token=token,
    uid=uid,
    sn="201607813254",  # 设备序列号
    start_time=start_time,
    end_time=end_time
)

# 4. 解析历史轨迹点
if result and isinstance(result, list) and len(result) > 0:
    device_info = result[0]
    history_list = device_info.get('historyList', [])
    
    print(f"找到 {len(history_list)} 个历史轨迹点")
    
    for i, point in enumerate(history_list, 1):
        print(f"点 {i}:")
        print(f"  纬度: {point.get('latitude')}")
        print(f"  经度: {point.get('longitude')}")
        print(f"  时间戳: {point.get('timestamp')}")
```

## 时间戳转换

### 将日期时间转换为毫秒时间戳

```python
from datetime import datetime

# 方法1: 使用 datetime
dt = datetime(2024, 1, 1, 0, 0, 0)  # 2024-01-01 00:00:00
timestamp_ms = int(dt.timestamp() * 1000)

# 方法2: 使用字符串解析
date_str = "2024-01-01 00:00:00"
dt = datetime.strptime(date_str, "%Y-%m-%d %H:%M:%S")
timestamp_ms = int(dt.timestamp() * 1000)
```

### 将毫秒时间戳转换为日期时间

```python
from datetime import datetime

timestamp_ms = 1609459200000
dt = datetime.fromtimestamp(timestamp_ms / 1000)
print(dt.strftime("%Y-%m-%d %H:%M:%S"))
```

## 响应数据结构

### 返回格式

返回的是一个数组，包含设备信息对象。每个设备对象包含：

```json
{
  "sn": "设备序列号",
  "name": "设备名称",
  "cid": "设备ID",
  "uid": "用户ID",
  "mac": "MAC地址",
  "status": "状态",
  "timestamp": "时间戳",
  "address": "地址",
  "lastAddress": "最后地址",
  "lastArea": "最后区域",
  "lastLatLng": "最后经纬度",
  "createtime": "创建时间",
  "updatetime": "更新时间",
  "bandtime": "绑定时间",
  "icon": "图标",
  "remarks": "备注",
  "shareFrom": false,
  "historyList": [  // 历史轨迹点数组
    {
      "latitude": 39.908823,
      "longitude": 116.397470,
      "timestamp": "1609459200000",
      "accuracy": "10",
      "confidence": "99",
      "datePublished": "2024-01-01 00:00:00",
      "updatetime": "2024-01-01 00:00:00",
      "latLngType": 0
    },
    // ... 更多轨迹点
  ]
}
```

### TrackData 字段说明

每个历史轨迹点（TrackData）包含以下字段：

- `latitude`: 纬度（double）
- `longitude`: 经度（double）
- `timestamp`: 时间戳（字符串，毫秒）
- `accuracy`: 精度（字符串）
- `confidence`: 置信度（字符串）
- `datePublished`: 发布日期（字符串）
- `updatetime`: 更新时间（字符串）
- `latLngType`: 经纬度类型（整数，0=高德地图，1=Google地图）

## 常见使用场景

### 场景1: 获取最近24小时的历史轨迹点

```python
from datetime import datetime, timedelta

end_time = int(datetime.now().timestamp() * 1000)
start_time = end_time - 24 * 60 * 60 * 1000

result = call_and_decrypt_api(
    token=token,
    uid=uid,
    sn="201607813254",
    start_time=start_time,
    end_time=end_time
)
```

### 场景2: 获取指定日期范围的历史轨迹点

```python
from datetime import datetime

# 2024年1月1日 00:00:00 到 2024年1月2日 23:59:59
start_date = datetime(2024, 1, 1, 0, 0, 0)
end_date = datetime(2024, 1, 2, 23, 59, 59)

start_time = int(start_date.timestamp() * 1000)
end_time = int(end_date.timestamp() * 1000)

result = call_and_decrypt_api(
    token=token,
    uid=uid,
    sn="201607813254",
    start_time=start_time,
    end_time=end_time
)
```

### 场景3: 获取今天的历史轨迹点

```python
from datetime import datetime, timedelta

today_start = datetime.now().replace(hour=0, minute=0, second=0, microsecond=0)
today_end = datetime.now()

start_time = int(today_start.timestamp() * 1000)
end_time = int(today_end.timestamp() * 1000)

result = call_and_decrypt_api(
    token=token,
    uid=uid,
    sn="201607813254",
    start_time=start_time,
    end_time=end_time
)
```

## 注意事项

1. **时间戳单位**: 必须使用**毫秒**时间戳，不是秒时间戳
2. **时间范围**: startTime 必须小于 endTime
3. **空结果**: 如果指定时间范围内没有轨迹点，`historyList` 将为空数组
4. **默认值**: 如果不提供 startTime 和 endTime，将使用默认值 1，可能不会返回历史轨迹点
